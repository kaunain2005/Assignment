How to test this project:

1. Start the server

python manage.py runserver

2. Open Postman and import the Postman collection

The machine_test.postman_collection.json file is included. It contains:

  • Get token (JWT)
  • Create, update, delete client (Manager only)
  • Add project and assign users (Manager only)
  • Get projects assigned to logged-in Developer

3. Use credentials

  • Manager: ganesh / managerpass
  • Developer: rohit / developerpass

4. Test API endpoints

All requests must use the Bearer token from the JWT login. For Manager, you can:

  • Register a client
  • Fetch all clients
  • Update or delete a client
  • Add projects and assign developers

Developers can:

  • View only their assigned projects
  • Cannot create/edit/delete clients or projects
